# Movies Catalog 🎬

Aplicação desenvolvida em React para demonstrar conceitos fundamentais da biblioteca, como componentização, props, gerenciamento de estado com useState e estilização com Styled Components.

## Funcionalidades

* Catálogo com 10 filmes
* Filtro por categoria
* Sistema de favoritos
* Tela exclusiva para filmes favoritados
* Componentes reutilizáveis
* Interface responsiva e moderna
* Navegação entre catálogo e favoritos
* Atualização dinâmica da interface através de estados

## Tecnologias Utilizadas

* React
* JavaScript (ES6+)
* Styled Components
* Vite

## Conceitos Aplicados

* Componentização
* Props
* useState
* Renderização dinâmica com map()
* Filtragem de dados
* Atualização de estado imutável
* Comunicação entre componentes
* Organização de projeto em páginas, componentes e dados
